Introduction

Dataset:

This dataset is downloaded at website https://www.kaggle.com/c/house-prices-advanced-regression-techniques .The goal of this data visualization is to find the relation between the sales price of a house and other variables, such as the total square feet, garage space, living space, number of rooms, and remodeled year. Moreover, this data visualization shows the price varying from 2006 to 2010. The utilized data is pre-processed using R and MATLAB offline to emphasize the relationships among the variables with the house price.

Visualization Structure

This narrative visualization follows an interactive slideshow, where the users are allowed to explore in some slides. Because there are multiple variables that affects the house price, selecting an interactive slideshow for visualization is adequate to observe difference between variables.

Scenes

Four scenes are included in this visualization using a consistent template and layout. The colors of each type of variable are consistent throughout the whole visualization. The slides can be navigated using arrows at left or right, or even using the dots on top. The visualization starts from observing variables from the big picture. Then, the variables are quantized for comparison and presented in different charts to observe the relationships.

Annotations

In each chart, orange-colored boxes and text is used to highlight the takeaway of the chart. For example, in the first chart, the orange box indicates that the houses are sold more during spring and summer compared to other months. If selecting a single variable using the radio button, then the annotations will indicate the overall relationship between the selected variable and the house price.

Parameters

First, in the first slide, the parameters are the slide number, which changes from 1 to 6 for each slideshow. In the third slide, the parameters are the slide number and the value of the radio button, which will change as the user select. The value will then change the plot to gradient colored image, showing the relationship between variables and house price. In the second chart (4th slide), additional parameters such as the number box is added for adjusting the number of bins. For the third chart, the button value will change the slideshow number after clicked.

Triggers

For the first chart, the radio buttons serve as filters to change the color of the data points. That is, the color will turn into different gradient colors corresponding to different variables. This will reveal the overall relationship between the selected variable and the house price. Moreover, automated animation of the annotation is triggered as the figures are completely loaded. In the second chart, the number of bins of the histogram can be adjusted to see how the number of houses sold during different months. Transitions will take place as the bins are lowered to a certain number of bins. For the third chart, a button is designed to convert the chart from Line Chart into Radar Chart. Finally, in the radar chart, the influence value of each varaible can be accessed by demand. A tooltip will show up as the mouse moves over the poins.
This narrative visualization was created and checked using Chrome. Thus, selecting Chrome is recommended.
